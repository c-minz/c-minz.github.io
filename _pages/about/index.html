Causal sets and Hasse diagrams

Alongside my doctoral studies, I developed a LaTeX package to draw Hasse diagrams for causal sets. It is available through CTAN, so that it already comes with a full LaTeX installation (for example, it is ready to use on Overleaf). Just load the package with

\usepackage{causets}

Package features are:

  • has short macros working in text and math mode
  • the macro input is a permutation of consecutive integers
  • generalises the idea of 2D-orders to draw generic diagrams
  • uses TikZ to draw diagrams for ease of use also in larger graphics
  • the online PrOSET Editor helps to generate the macro (currently working for 2D-orders): PrOSET Editor
